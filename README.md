# Registro Carburante — MG ZS Hybrid+

App personale per tracciare consumi e spese carburante con metodo "livello costante".

## Stack
- React + Vite
- Supabase (PostgreSQL)
- Recharts (grafici)
- GitHub Pages (hosting)

## Sviluppo locale
```bash
npm install
npm run dev
```

## Deploy
Push su `main` → GitHub Action builda e deploya su GitHub Pages automaticamente.
